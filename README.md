# java-cli-gradle-spring-mockito-cucumber-junit-car-data

## Description
A POC for spring app. Testing done with
cucumber and mockito framework.

## Tech stack
- java
- gradle
  - spring
  - cucumber
  - mockito
    - junit5

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
